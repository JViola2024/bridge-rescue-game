# Bridge Rescue Game

Ez a projekt egy egyszerű 2D-s oktatási játék, ami Godot Engine-ben készült.

A játék elején a felhasználó témát választhat:
- Irodalom
- Történelem
- Informatika

A játék során kérdések jelennek meg, amikre négy válaszlehetőség közül lehet választani. Helyes válasz esetén a híd épül, rossz válasz esetén pedig a híd egy eleme eltűnik.

A játékosnak 24 kérdés áll rendelkezésére, hogy felépítse a 12 elemből álló hidat. Ha sikerül befejezni a hidat, a lovag automatikusan eljut a hercegnőhöz.

A projekt GDScript nyelven készült.

## Futtatás

A projekt a `project.godot` fájl megnyitásával indítható el Godot Engine-ben.
